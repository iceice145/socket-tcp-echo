# Python 并发 TCP Echo 服务
一个极简的 Python Socket 编程实现，基于 TCP 协议构建并发 Echo 服务（客户端发什么，服务器回什么），支持多客户端同时连接，核心代码仅 50 余行。

## 功能说明
- **并发支持**：服务器通过多线程处理客户端连接，可同时响应多个客户端
- **双向交互**：客户端实时发送消息，服务器即时回声回复
- **轻量无依赖**：仅使用 Python 标准库（`socket` + `threading`），无需额外安装依赖

## 环境要求
- Python 3.11（兼容 3.x 所有版本）
- 操作系统：Windows 10
